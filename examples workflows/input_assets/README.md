# Input Assets

This folder contains public demo images used by the maintained example workflows.

Files:

- `cat_announcer.png`
- `racecar_start.jpg`
- `racecar_end.jpg`

Usage:

1. Copy the file you need into your ComfyUI `input` directory.
2. Import the matching workflow JSON.
3. If you prefer, replace the bundled images with your own media.
